# Multi-Image Bot


### How to start it

1) Setup a token as environmental variable BOT_TOKEN
```bash
export BOT_TOKEN=<your token>
```
2) install requirements from requirements.txt

3) Download model checkpoint from [https://drive.google.com/drive/folders/1YndQnNhYAUfTT5I0r2H2ozey7T7GkHh-?usp=sharing](link)
4) Place the checkpoint inside /bot/model_ckpt/
5) Run bot.py
